rpm2cpio

Standalone RPM to CPIO utility written in Python.

Download this project as a .zip file Download this project as a tar.gz file

rpm2cpio utility converts RPM on standard input (or as the first parameter) to a CPIO archive on standard output, so you can pipe it through CPIO.

Usage:

rpm2cpio < adjtimex-1.20-2.1.i386.rpm  | cpio -it
./sbin/adjtimex
./usr/share/doc/adjtimex-1.20
./usr/share/doc/adjtimex-1.20/COPYING
./usr/share/doc/adjtimex-1.20/COPYRIGHT
./usr/share/doc/adjtimex-1.20/README
./usr/share/man/man8/adjtimex.8.gz
133 blocks
